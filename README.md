# ATM Interface System (Java)

A console-based Java application that simulates an ATM system with secure PIN authentication, balance inquiry, deposits, withdrawals, and transaction history management.

## Overview
This project was built to practice object-oriented programming, problem solving, and design pattern implementation using Java.

The application allows users to simulate ATM interactions by inserting a card, entering a PIN, performing banking operations, viewing transaction history, and securely ending sessions through a menu-driven console interface.

## Features
- Secure PIN authentication system
- ATM state management using the State Design Pattern
- Balance inquiry functionality
- Cash deposit feature
- Cash withdrawal with insufficient balance handling
- Transaction history tracking
- Input validation for numbers and amounts
- Multiple bank account support
- Menu-driven console interface
- Session handling and card ejection simulation

## How to Run the Project
1. Download or clone the repository
2. Open the project in IntelliJ IDEA
3. Run the `Main.java` file
4. Enter the test PIN when prompted
5. Select operations from the ATM menu
6. View balances, perform transactions, and check transaction history in the console
